Conhecimentos

HTML

O HTML5 é a versão recente da linguagem de programação conhecida como HTML (Hyper Text Markup Language), sigla em Inglês para Linguagem de Marcação de Hipertexto. Essa linguagem usa um texto codificado para criar e organizar conteúdo online.

CSS

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet)..

Design

Web Designer é uma ferramenta visual, mas o usuário também pode utilizar JavaScript e CSS para deixar a criação ainda mais personalizada. ...

Javascript

JavaScript é uma linguagem de programação que permite você implementar itens complexos em páginas web, toda vez que uma página web faz mais do que simplesmente mostrar a você informação estática, mostrando conteúdo que se atualiza.

Boostrap

Bootstrap é um framework front-end que facilita a vida dos desenvolvedores web a criar sites com tecnologia mobile (responsivo) sem ter que digitar uma linha de CSS para “fazer e acontecer”.

Git e GitHub

git e github é que o git é só uma ferramenta para versionar projetos, enquanto o github é o site no qual você colocará esses projetos versionados. uma analogia válida seria que o git é seu pincel e tintas enquanto o github é um museu.
